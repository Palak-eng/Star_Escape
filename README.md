# 🎮 Star Escape Game

A 2D Unity game where the player collects stars while avoiding enemies.

## Features
- Player movement (WASD)
- Enemy AI (Patrol + Chase)
- Score system
- High Score system
- Start Menu
- Pause Menu
- Game Over + Win Screen
- Sound effects + BGM

## Controls
- W A S D → Move
- ESC → Pause
